# User Guide: Layer Permissions

## Layer Permission Types

Each layer can be either **Public** or **Restricted**.

### Public Layers
- Anyone with an account can view and download
- No special permission needed

### Restricted Layers
- Only users with permission can view and download
- Contact your administrator to request access
- Marked with a restricted icon (🚫)

## Permission Levels

### No Permission (Restricted layers only)
- ✅ See layer name in lists
- ❌ View details
- ❌ Download files

### Read Permission
- ✅ View layer details
- ✅ Download files
- ❌ Edit or upload

### Read and Write Permission
- ✅ View and download
- ✅ Edit layer information
- ✅ Upload new versions

### Administrator
- ✅ Full access to all layers (public and restricted)

## How to Request Access

To access a restricted layer:

1. Contact your system administrator
2. Provide the layer name
3. Specify whether you need Read or Read and Write access
